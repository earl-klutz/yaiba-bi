# YAIBA-BIとは
YAIBA-BIは、VRChatで取得した位置情報ログから、
**誰でも簡単に人流解析・ヒートマップ・可視化動画を生成できるツール**です。
本来 YAIBA はプログラマー向けのツールですが、
YAIBA-BI では **Google Colab にログを入れるだけ**で
自動でグラフや動画がダウンロードできることを目指しています。
